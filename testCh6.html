第六章 問題：法律、風險與合規性（含答案與說明）

1. 您的公司正在考慮將其生產環境遷移到雲端。在審查提議的合同時，您注意到其中包含一條條款，要求在預定日期之前任何時間終止合同，需支付額外費用，相當於六個月的付款（等於合同期限的一半）。這最好被描述為 _________________ 的例子。

  1. 有利的合同條款
  2. 強勢的談判
  3. 基礎設施即服務 (IaaS)
  4. 供應商鎖定

答案： D

說明： 供應商鎖定發生在客戶因故不願離開供應商，即使這是最佳決策時。這些合同條款只能從供應商的角度被描述為有利；選項 D 比選項 A 更適合描述這種情況。問題中沒有包含談判的描述；選項 B 不正確。IaaS 是一個服務模型，與此上下文中的任何內容都不真正相關；選項 C 不正確。

2. Cathy 正在開發一個電子發現計劃，以幫助她的組織正式確定其遵守法律保留義務。她希望使用行業標準來指導她實現最佳實踐。她應考慮使用哪個標準來進行這項工作？

  1. ISO 27001
  2. ISO 27002
  3. ISO 27050
  4. ISO 27701

答案： C

說明： ISO 27050 是一個為電子發現計劃提供指導的行業標準。ISO 27001 和 ISO 27002 提供行業標準的控制目標和網路安全控制建議。ISO 27701 為資訊隱私計劃提供行業標準指導。

3. 關於大多數隱私指南，數據處理者是 _________________。

  1. 個人身份資訊 (PII) 所描述的個人
  2. 收集或創建個人身份資訊 (PII) 的實體
  3. 代表控制者使用個人身份資訊 (PII) 的實體
  4. 監管個人身份資訊 (PII) 的實體

答案： C

說明： 代表所有者/控制者使用數據的實體是數據處理者。數據主體是個人身份資訊 (PII) 所描述的人。收集或創建 PII 的實體是數據所有者或控制者。監管 PII 使用的實體是監管機構。

4. 您的公司在一場涉及違約的民事訴訟中為自己辯護。您的 IT 部門人員對反映與案件相關情況的事件日誌進行了鑑識分析。
為了讓您的人員能夠在鑑識分析期間作為專家證人呈現他們收集的證據，您應確保 _________________。

  1. 他們的證詞是經過編寫的，並且他們不偏離腳本
  2. 他們只呈現對您方有利的證據
  3. 他們接受過他們使用的工具的培訓並獲得了認證
  4. 他們在法庭作證期間按時獲得報酬

答案： C

說明： 為了提供可信、可信的專家證詞，您的員工對他們用於執行分析的任何鑑識工具都應具有不僅僅是業餘的理解和熟悉程度。正式培訓和認證是建立可信度的極佳方法。編寫證詞通常會受到法庭的鄙視；指導證人如何在法庭上表現以及預期會發生什麼是可以的，但這並不能帶來可信度。選項 A 不正確。您的專家證人不被允許從他們的證詞中隱瞞任何與案件相關的證據，即使該證據有助於對方。選項 B 不正確。您應該為您的員工支付他們的時間，無論他們是在工作現場還是在法庭上，但這與提高可信度無關。選項 D 不正確。

5. 在對其組織進行定性風險評估後，Prisha 決定建議添加一個新的防火牆模塊，該模塊將過濾掉入站惡意軟體。她正在推薦哪種類型的風險響應行為？

  1. 接受
  2. 轉移
  3. 降低
  4. 拒絕

答案： C

說明： 部署防火牆是一種旨在降低風險可能性或影響的風險緩解策略。如果 Prisha 建議組織繼續按原樣運行，那將是風險接受。風險轉移會將風險轉移給第三方，例如保險提供商。拒絕或否認風險不是一種有效的策略，儘管它確實會發生！

6. Nora 是 Acme Widgets 的一名員工，她是一個審計團隊的成員，該團隊檢查組織的財務控制。她目前正在參與一個評估向雲提供商付款是否適當的項目，並將向管理層報告她的結果。哪個術語最能描述 Nora 在此項目中的角色？

  1. 內部評估
  2. 外部審計
  3. 內部審計
  4. 外部評估

答案： C

說明： Nora 是該組織的員工，因此她的工作顯然是內部性質的。外部工作由獨立的第三方執行。Nora 是一名審計員，她正在測試控制的有效性，因此她的工作屬於正式審計的範圍，而不是非正式評估。因此，這個項目應被描述為內部審計。

7. Carla 被指派管理她組織的隱私計劃，並正在努力就組織隱私實踐的變更與客戶溝通。她計劃發送一封電子郵件通知客戶變更，並允許他們選擇退出對其數據的使用。此場景中未描述哪個 GAPP 原則？

  1. 通知
  2. 管理
  3. 訪問
  4. 選擇和同意

答案： C

說明： Carla 被指派管理其組織的隱私計劃。這項指派是管理 GAPP 原則的一個例子。她正在與客戶溝通關於隱私實踐的變更，這是通知的一個例子。她還向這些客戶提供選擇退出對其數據使用的機會。這是選擇和同意原則的一個例子。需要注意的是，同意不必是明確的並且在選擇加入的基礎上進行。選擇退出、隱含同意也滿足此原則。訪問原則規定個人應能夠審查和更新其個人資訊。在此場景中沒有描述訪問。

8. 您是美國中西部一所私立研究型大學的醫學生；您通過借記卡直接從您的銀行帳戶支付學費。以下哪個法律和標準將 *不* 適用於您、您的個人數據或您作為學生使用的數據？

  1. 薩班斯-奧克斯利法案 (SOX)
  2. 健康保險可攜性與責任法案 (HIPAA)
  3. 支付卡行業數據安全標準 (PCI DSS)
  4. 家庭教育權利和隱私法案 (FERPA)

答案： A

說明： SOX 僅適用於上市公司，不適用於所有公司。如果您處理患者數據，HIPAA 可能適用於您作為醫學生使用的數據。您的付款和個人數據受 PCI DSS 管轄。FERPA 保護您的個人學生信息。

9. Rolando 是一家大型雲服務提供商的風險經理。該公司最近評估了加利福尼亞泥石流對其在該地區運營的風險，並確定響應成本超過了其可以實施的任何控制措施的好處。該公司選擇此時不採取任何行動。Rolando 的組織推行了哪種風險管理策略？

  1. 風險規避
  2. 風險緩解
  3. 風險轉移
  4. 風險接受

答案： D

說明： 在風險接受策略中，組織決定不採取任何行動是管理風險的最有益途徑。

10. Yolanda 是一家金融機構的首席隱私官，正在研究與客戶支票帳戶相關的隱私要求。以下哪個法律最有可能適用於這種情況？

  1. GLBA
  2. SOX
  3. HIPAA
  4. FERPA

答案： A

說明： 格雷姆-里奇-比利勒法案 (GLBA) 包含規範客戶金融資訊隱私的規定。它特別適用於金融機構。

11. Bill 正在根據 SSAE 和 ISAE 標準對雲提供商進行審計。在審計期間，他發現完成其一項測試所需的某些記錄被意外銷毀且無法恢復。此控制目標沒有替代測試可用。Bill 應採取什麼行動？

  1. 在審計範圍說明中描述限制。
  2. 將審計推遲一年，直到有足夠的記錄可用。
  3. 出具不合格的審計報告。
  4. 從審計中移除此測試並測試不同的控制目標。

答案： A

說明： 在記錄不可用時的正確做法是編寫一份範圍限制說明，描述問題及其對審計的影響。Bill 本可以通過對同一控制目標執行替代測試來避免這種情況，但場景說明這是不可能的。

12. 以下哪項 *不是* 允許位於歐盟 (EU) 之外的實體收集和處理屬於歐盟公民的隱私數據的方式？

  1. 位於擁有符合歐盟法律的全國性法律的國家。
  2. 向歐盟高等法院上訴以獲得許可。
  3. 創建符合歐盟法律的具有約束力的合同語言。
  4. 在其自己的國家加入隱私護盾計劃。

答案： B

說明： 通用數據保護條例禁止在沒有全國性隱私法的國家內的實體收集或處理屬於歐盟公民的隱私數據。如果滿足以下條件，則允許實體這樣做：(1) 其自己的國家擁有符合歐盟法律的全國性法律。(2) 該實體創建符合歐盟法律的合同語言，並獲得該實體希望收集公民數據的每個歐盟國家的批准。(3) 該實體自願訂閱其自己國家的隱私護盾計劃（假設該計劃被歐盟當局認為可接受）。然而，沒有程序允許該實體向歐盟上訴以獲得許可。

13. 在嘗試評估故障對客戶信心的影響時，哪種類型的業務影響評估工具最合適？

  1. 定量分析
  2. 定性分析
  3. 年度損失預期
  4. 減少

答案： B

說明： 定性工具通常用於業務影響評估中，以捕捉對無形因素（如客戶信心、員工士氣和聲譽）的影響。

14. 對 _________________ 的審計將證明組織擁有一個全面的、整體的內部安全控制計劃。

  1. 審計標準聲明 (SAS) 70 標準
  2. 鑒證業務標準聲明 (SSAE) 18 標準
  3. 服務組織控制 (SOC) 2，類型 2 報告矩陣
  4. ISO 27001 認證要求

答案： D

說明： ISO 27001 認證適用於資訊安全管理系統 (ISMS)，即組織的整個安全計劃。SAS 70 和 SSAE 18 是服務提供商的審計標準，包括一些安全控制審查，但不是一個有凝聚力的計劃（並且 SAS 70 已過時）；選項 A 和 B 不正確。SOC 報告是 SSAE 18 審計的執行方式；選項 C 不正確。

15. IT 安全審計旨在揭示以下所有內容，*除了* _________________。

  1. 財務欺詐
  2. 故障控制
  3. 控制不足
  4. 未能達到目標標準和指南

答案： A

說明： IT 安全審計無意發現財務欺詐；然而，它可能會無意中導致此類揭露。存在專門用於此目的的其他特定審計。所有其他選項都是不正確的，因為它們是 IT 安全審計的預期目標。

16. 在 IT 審計期間，雲提供商的 CEO 要求定期更新測試過程。審計師應如何回應此要求？

  1. 拒絕向 CEO 提供任何資訊，直到審計結束。
  2. 將此事提交給客戶的董事會。
  3. 向 CEO 提供定期更新。
  4. 將此事提交給審計公司的合夥人審查委員會。

答案： C

說明： 在審計過程中與利益相關者接觸是完全恰當的。雖然 CEO 可能以粗魯的方式要求提供資訊，但這並不意味著他們不是重要的利益相關者。審計團隊應仔細接觸 CEO，並在整個審計過程中讓他們保持參與。只有在 CEO 提出不當請求時，才需要將此事提交給其他機構。

17. 以下哪個是常用於評估雲提供商的美國審計標準？

  1. ISO 27001
  2. SOX
  3. SSAE 18
  4. IEC 43770

答案： C

說明： 鑒證業務標準聲明 (SSAE) 18 是當前的 AICPA（美國註冊會計師協會）審計標準。ISO 27001 是一個國際審計標準。薩班斯-奧克斯利法案 (SOX) 是一項涉及上市公司的美國法律。IEC 43770 標準不存在。

18. 數字鑑識調查人員通常執行以下所有操作，*除了* 安全地 _________________ 數據。

  1. 收集
  2. 創建
  3. 分析
  4. 呈現

答案： B

說明： 除了極少數例外，數字鑑識不包括創建數據（鑑識報告中關於分析的數據除外）。雖然這也可以被認為是數字鑑識的一個方面，但其他選項更適合描述數字鑑識，因此這是最佳的否定答案。

19. A(n) _________________ 包括根據給定標準審查組織的當前狀況/績效。

  1. 服務組織控制 (SOC) 報告
  2. 差距分析
  3. 審計範圍說明
  4. 聯邦指南

答案： B

說明： 這就是差距分析的定義。SOC 報告是特定類型的審計；選項 A 不正確。範圍說明是一個審計前功能，有助於組織和審計員確定具體將審計什麼。選項 C 不正確。聯邦指南是關於應如何做某事的政府建議。選項 D 不正確。

20. Belinda 正在審計一家製造公司的財務控制，並了解到財務系統運行在一個主要的 IaaS 平台上。她希望獲得該平台具有適當安全控制的保證，以確保客戶財務報表的準確性。她應採取什麼行動？

  1. 對雲提供商執行 IT 審計。
  2. 獲取 SOC 1 報告。
  3. 獲取 SOC 2 報告。
  4. 繼續僅測試客戶處的控制，並在其報告中註明雲提供商的使用。

答案： B

說明： Belinda 有義務確保雲提供商具有適當的控制措施。她不太可能獲得審計這些控制措施的許可，即使她獲得了此許可，也會導致過多且不必要的成本。相反，她應向雲提供商索取獨立審計報告。SOC 1 審計專門設計用於測試涵蓋客戶財務報表的控制措施，將是此場景中最合適的審計類型。SOC 2 審計更廣泛地涵蓋網路安全控制，是不必要的。

21. Tony 正在制定業務連續性計劃，但在優先級排序資源方面遇到困難，因為難以結合有關有形和無形資產的資訊。對他來說，最有效的風險評估方法是什麼？

  1. 定量風險評估
  2. 定性風險評估
  3. 既非定量也非定性風險評估
  4. 定量和定性風險評估的組合

答案： D

說明： Tony 通過結合定量和定性風險評估的元素將看到最佳結果。定量風險評估擅長分析財務風險，而定性風險評估是處理無形風險的好工具。結合這兩種技術可以提供全面的風險圖像。

22. 哪個是第一個專門針對雲提供商的國際隱私標準？

  1. 國家標準與技術研究院 (NIST) 特別出版物 (SP) 800-37
  2. 個人資訊保護和電子文件法案
  3. 支付卡行業
  4. ISO 27018

答案： D

說明： ISO 27018 描述了雲提供商的隱私要求，包括年度審計要求。選項 A 不正確，因為 NIST SP 800-37 描述了風險管理框架，不是國際隱私標準。個人資訊保護和電子文件法案是一項涉及數據隱私的加拿大法律。選項 B 不正確。選項 C 不正確，因為 PCI DSS 專門針對接受信用卡的商家，而不是雲提供商（雖然雲提供商可能處理信用卡，因此必須遵守 PCI DSS，但選項 D 是更可取、更好的答案）。

23. 以下哪個資訊元素 *不* 被視為會觸發大多數美國州數據洩露法律的直接標識符？

  1. 學生身份證號碼
  2. 社會安全號碼
  3. 駕照號碼
  4. 信用卡號碼

答案： A

說明： 大多數州數據洩露法律模仿加利福尼亞州的數據洩露通知法，涵蓋社會安全號碼、駕照號碼、州身份證號碼、信用卡/借記卡號碼以及銀行帳號（與 PIN 或密碼結合）。加利福尼亞州的洩露通知法還保護一些在其他州法律中不常見的項目，包括醫療記錄和健康保險信息。這些法律與諸如加利福尼亞消費者隱私法案 (CCPA) 之類的隱私法律是分開且不同的，後者更廣泛地規範個人信息的處理。

24. 在雲客戶和雲提供商之間的合同中包含的以下哪個項目最能幫助減少供應商鎖定？

  1. 數據格式類型和結構
  2. 可用性
  3. 儲存空間
  4. 可用操作系統列表

答案： A

說明： 當雲客戶可以確保其數據不會被移植到專有數據格式或系統時，客戶更有保證不會被限制在給定提供商處；平台無關的數據集更具可移植性，更不容易受到供應商鎖定的影響。可用性可能是可移植性的一個方面；客戶訪問自己數據的便捷性和速度會影響數據轉移到另一個提供商的難易程度。然而，這不如數據的格式和結構影響大；選項 A 優於選項 B。儲存空間與供應商鎖定關係不大；選項 C 不正確。提供商提供的操作系統列表可能會影響客戶選擇哪個提供商的決定，但通常不是會限制可移植性的限制因素。選項 D 不正確。

25. 以下哪個合同條款最能激勵雲提供商滿足服務級別協議 (SLA) 中列出的要求？

32. Viola 正在計劃進行用戶帳戶審計，以確定帳戶是否具有適當級別的權限，以及所有權限是否都通過正式流程批准。該組織大約有 50,000 個用戶帳戶，年員工流失率為 24%。在選擇記錄進行手動審查時，以下哪個抽樣方法對她的時間利用最有效？

63. _________________ 是法律概念，即雲客戶被要求對提供其用戶和客戶隱私數據的安全性負有合理期望。

說明： 數據保管人角色分配給負責實施由策略和高級管理層定義的安全控制的個人。數據所有者對這些任務負有最終責任，但數據所有者通常是高級領導者，將運營責任委派給數據保管人。

78. A _____________ 通常採用一套基於絕對數值評估風險的方法、原則或規則。

79. Nolan 是一位美國聯邦政府機構的採購官，正在選擇一家雲服務提供商。哪個計劃提供了一套經預先篩選、授權在聯邦政府使用的雲提供商？

80. 在收集可能在法庭上使用的安全事件證據時，組織應遵循哪些程序？

82. 您希望確保您的組織的保險單能夠充分覆蓋安全事件造成的損害，以便您恢復運營。您應使用哪種資產評估技術？

  2. 原始成本
  3. 估計
  4. 重置成本

答案： D

說明： 重置成本技術按當前市場上替換資產的價格對資產進行估值，是尋求覆蓋成本時最合適的技術。原始成本技術使用設備的購買價格。折舊價值技術採用原始成本並在其預期壽命內降低它。估計只是對資產價值進行有根據的猜測。

83. 您擔心如果雲供應商停止運營，您可能無法再訪問必要的源代碼。哪個安全控制最能防範此風險？

  1. 合同條款
  2. 託管
  3. SLA
  4. 訴訟

答案： B

說明： 如果您的雲供應商倒閉，您與他們簽訂的任何法律和合同條款基本上都將無效。因此，您不應依賴合同條款、服務級別協議 (SLA) 或訴訟來解決此問題。託管將代碼副本置於獨立第三方手中，如果供應商倒閉，該第三方將向客戶發布代碼。

84. 哪種類型的評估採用一套基於非數字類別或級別評估風險的方法、原則或規則？

  2. 定性評估
  3. 混合評估
  4. SOC 2

答案： B

說明： 定性評估是一套基於非數學或類別或級別評估風險的方法或規則。使用這些數學類別或級別的評估稱為定量評估。不存在混合評估這種東西，SOC 2 是一個關於控制有效性的會計報告。

85. 以下哪個術語通常 *不* 在雲服務提供商合同中找到？

  2. 審計權
  3. 終止條款
  4. 訪問數據的權利

86. Ben 正在尋找一個在世界範圍內被廣泛接受並專門側重於資訊安全控制的控制目標框架。以下哪個框架最能滿足他的需求？

89. Ron 是一家美國公司的 CISO，該公司正與一家歐洲公司建立業務夥伴關係。歐洲公司將向他的公司發送客戶記錄，以通過 Ron 公司專有的信用評分算法運行。根據 GDPR，Ron 的公司相對於客戶數據將扮演什麼角色？

  2. 數據所有者
  3. 數據主體
  4. 數據處理者

說明： 在這種情況下，Ron 的公司是數據處理者，因為它正在接收來自歐洲公司的記錄。在這種情況下，歐洲公司是數據控制者，因為它們對數據負責。記錄中描述的個人是數據主體。數據所有者負責決定數據（例如誰接收訪問權限以及如何使用數據）。

90. 以下哪個通常被視為供應鏈風險？（選擇所有適用項。）

  1. 在運送給最終客戶之前，對手篡改硬體
  2. 對手入侵組織在 IaaS 環境中運行的 Web 伺服器
  3. 對手使用社會工程手段入侵 SaaS 供應商的員工以獲取客戶帳戶的訪問權限
  4. 對手使用殭屍網路進行拒絕服務攻擊

答案： A, C

說明： 供應鏈風險發生在對手干擾從供應商到客戶的商品或服務交付時。這可能涉及在客戶收到硬體之前對其進行篡改，或使用社會工程手段入侵供應商員工。入侵在 IaaS 環境中運行的 Web 伺服器不是供應鏈風險，因為 Web 伺服器已在客戶控制之下。使用殭屍網路進行拒絕服務攻擊不涉及任何供應鏈元素。

91. 根據此場景中的資訊，龍捲風對 Atwood Landing 數據中心的影響的暴露因子是多少？（場景：Henry 是美國中西部 SaaS 提供商 Atwood Cloud Services 的風險經理。該公司的主要數據中心位於印第安納州北部一個易發龍捲風的地區。Henry 最近進行了一次重置成本分析，確定重建和重新配置數據中心的成本為 1000 萬美元。Henry 諮詢了龍捲風專家、數據中心專家和結構工程師。他們共同確定，一次典型的龍捲風會對設施造成約 500 萬美元的損壞。氣象學家確定 Atwood 的設施位於一個可能每 200 年經歷一次龍捲風的地區。）

  1. 10%
  2. 25%
  3. 50%
  4. 75%

答案： C

說明： 暴露因子是風險管理人員預期如果風險發生時設施損壞的百分比。它的計算方法是用損壞金額除以資產價值。在這種情況下，即 500 萬美元的損壞除以 1000 萬美元的設施價值，即 50%。

92. 根據此場景中的資訊，Atwood Landing 數據中心發生龍捲風的年度發生率是多少？（場景：參考問題 91 的場景）

  1. 0.0025
  2. 0.005
  3. 0.01
  4. 0.015

答案： B

說明： 年度發生率是風險分析師預期在任何給定年份風險發生的次數。在這種情況下，分析師預計每 200 年發生一次龍捲風，即每年 0.005 次。

93. 根據此場景中的資訊，Atwood Landing 數據中心發生龍捲風的年度損失預期是多少？（場景：參考問題 91 的場景）

  2. $50,000
  3. $250,000
  4. $500,000

答案： A

說明： 年度損失預期是通過將單次損失預期 (SLE) 乘以年度發生率 (ARO) 來計算的。在這種情況下，SLE 為 5,000,000 美元，ARO 為 0.005。將這些數字相乘得到 ALE 為 25,000 美元。

99. 您是一家大型醫院系統的 CISO，正在準備與一家軟體即服務 (SaaS) 電子郵件供應商簽訂合同，並希望執行控制評估以確保其業務連續性規劃措施是合理的。您可能會請求哪種類型的審計來達到此目標？

100. 以下哪個最不可能適合包含在雲客戶和雲提供商之間的服務級別協議 (SLA) 中？

  2. 管轄權
  3. 儲存空間
  4. 可用性
