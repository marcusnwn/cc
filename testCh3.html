第三章 問題：雲平台與基礎設施安全（含答案與說明）

1. Barry 是一家組織的 CIO，該組織最近遭遇了嚴重的運營問題，需要啟動災難恢復計劃。他想召開一次經驗教訓會議來審查這次事件。誰是這次會議的最佳主持人？

  1. Barry，作為首席資訊官
  2. 首席資訊安全官
  3. 災難恢復團隊負責人
  4. 外部顧問

答案： D

說明： Barry 應招募一位獨立的主持人來主持會議。讓一位未直接參與此次工作的外部主持人可以鼓勵誠實和開放的反饋。雖然不一定需要使用外部顧問，但他們可以輕鬆地擔任此角色。也可以找到合格的內部員工來擔任此職位，但不應是參與事件響應工作或在計劃中有重大利益關係的人，例如 Barry、CISO 或 DR 團隊負責人。

2. 如果雲用於 BC/DR 目的，失去 ___________________ 可能會嚴重影響您組織的 RTO。

  1. 任何雲管理員
  2. 特定的 VM
  3. 您的政策和合同文件
  4. ISP 連接性

答案： D

說明： 沒有 ISP 連接性，任何人都無法使用網際網路，因此也無法使用雲。當然，現實情況是，沒有網際網路連接，大多數組織無論是在雲端還是在本地運營，都無法完成太多業務。失去任何單一雲管理員不太可能嚴重影響您組織的 RTO。失去特定的 VM 可能不會嚴重影響您組織的 RTO。VM 可以輕鬆地重新實例化。選項 B 不正確。失去您的政策和合同文件不會嚴重影響您組織的 RTO。選項 C 不正確。

3. Brent 正在審查將保護其組織在其本地數據中心發生持續停電事件時的控制措施。以下哪個解決方案最能滿足他的需求？

  1. 冗餘伺服器
  2. 不間斷電源 (UPS)
  3. 發電機
  4. RAID

答案： C

說明： 發電機能夠在停電事件中提供持續的備用電源，但它們需要時間才能啟動。不間斷電源 (UPS) 提供即時的、由電池驅動的電源，用於覆蓋短暫的電力損失，但無法覆蓋持續的電力損失。廉價磁碟冗餘陣列 (RAID) 和冗餘伺服器是高可用性控制，但不涵蓋電力損失場景。

4. Carolyn 擔心她網路上的用戶可能未經適當授權或安全控制就在他們的硬碟上儲存敏感資訊，例如社會安全號碼。她可以實施哪個第三方安全服務來最好地檢測此活動？

  1. IDS
  2. IPS
  3. DLP
  4. TLS

答案： C

說明： 數據丟失預防 (DLP) 系統可以識別儲存在端點系統上或在網路上傳輸的敏感資訊。這是它們的主要目的。DLP 系統通常作為第三方託管服務提供。入侵檢測和預防系統 (IDSs/IPSs) 可能會使用為此目的構建的簽名來識別一些敏感資訊，但這不是這些工具的主要作用，它們在此任務上的效果不如 DLP 系統。TLS 是一種網路加密協議，可用於保護敏感資訊，但它沒有任何識別敏感資訊的能力。

5. 哪些個人應該有權訪問雲數據中心的管理平面？

  1. 服務提供商工程師
  2. 客戶工程師
  3. 終端用戶
  4. A 和 B

答案： A

說明： 雲服務提供商數據中心的管理平面應保留給該提供商自己的工程師使用。管理平面上的流量控制著基礎設施本身的運行，授予客戶（即使是訓練有素的工程師）訪問該網路可能會危及其他客戶的安全。

6. Roland 是一家組織的物理安全專家，該組織擁有大量經常在設施內移動的昂貴實驗室設備。以下哪種技術能以最具成本效益的方式提供庫存控制過程的最大自動化？

  1. IPS
  2. Wi-Fi
  3. RFID
  4. 乙太網

答案： C

說明： 無線射頻識別 (RFID) 技術是一種追蹤設施內物品的經濟有效方式。雖然 Wi-Fi 可用於相同目的，但其實施成本要高得多。

7. Becka 最近簽訂了一份合同，內容是關於一個備用數據處理設施，該設施將在發生災難時為她的公司提供空間。該設施包括 HVAC、電力和通信線路，但沒有硬體。Becka 使用的是哪種類型的設施？

  1. 冷站
  2. 暖站
  3. 熱站
  4. 移動站

答案： A

說明： 冷站包括數據中心運營所需的基本能力——空間、電力、HVAC 和通信——但不包括恢復運營所需的任何硬體。

8. 您的組織將其生產環境託管在雲環境中。您正在考慮也將雲備份服務用於您的 BC/DR 目的。就冗餘和彈性而言，這種方法的最佳策略可能是什麼？

  1. 讓您的雲提供商也提供 BC/DR 備份。
  2. 在您公司總部的場所保留 BC/DR 備份。
  3. 使用另一個雲提供商進行 BC/DR 備份。
  4. 將您的生產環境移回您的公司場所，並使用您的雲提供商託管您的 BC/DR 備份。

答案： C

說明： 最好在另一個雲提供商處進行備份，以防任何導致服務中斷的原因發生在您的主要提供商的整個環境中；這將更加複雜和昂貴，但它提供了最佳的冗餘和彈性。使用相同的提供商進行生產和備份不是一個壞選擇，但它帶來了相同的應急情況影響數據兩個副本的風險。將備份或生產環境本地化都不能提供最佳保護，因此選項 B 和 D 都不是可取的。

9. Philip 正在開發一個新的安全工具，該工具將被他組織的許多不同子公司的個人使用。他選擇使用 Docker 來部署該工具以簡化配置。哪個術語最能描述這種方法？

  1. 虛擬化
  2. 抽象化
  3. 簡化
  4. 容器化

答案： D

說明： 所有這些術語都準確地描述了這種技術的使用。然而，Docker 的使用最好被描述為容器化技術，所以這是最佳的答案選擇。

10. 在雲 BC/DR 活動中，需要保護的最重要資產是什麼？

  1. 知識產權
  2. 雲數據中心的硬體
  3. 人員
  4. 可攜式媒體上的數據

答案： C

說明： 在所有安全活動中，健康和人類安全始終是至關重要的。所有這些資產都需要某種類型的保護；然而，人類安全必須始終是最高優先級。

11. Carla 正在開發一個雲基礎設施服務產品的設計，她將把這個產品轉售給一些客戶。她的堆疊中哪個組件最直接負責執行屬於不同客戶的虛擬機的租戶分區？

  1. 訪問控制列表
  2. 網路安全組
  3. 防火牆
  4. Hypervisor

答案： D

說明： 所有這些技術都在租戶分區中扮演一定角色。然而，這個問題特別詢問了屬於不同租戶的虛擬機的分區。這是虛擬化平台上 hypervisor 的責任。

12. Carlos 正在為一個將建在新的四層公司總部內的數據中心進行設計規劃。該建築包括一個地下室和三個地上樓層。數據中心的最佳位置是哪裡？

  1. 地下室
  2. 一樓
  3. 二樓
  4. 三樓

答案： C

說明： 數據中心應位於建築物的核心。將其置於地下室使其容易受到洪水侵襲。一樓是建築物的正常入口點，使其更容易受到物理闖入。將其置於頂樓使其容易受到風和屋頂損壞。

13. Chris 是一家大型公司的資訊安全專業人員，當他走進大樓時，他注意到一個安全區域的門被敞開了。物理安全不屬於他的職責範圍，但他立即採取行動關上門，並通知物理安全團隊他的行動。Chris 正在展示哪個原則？

  1. 盡職調查 (Due care)
  2. 審慎查核 (Due diligence)
  3. 職責分離
  4. 知情同意

答案： A

說明： 盡職調查原則規定，個人應在情況下使用任何理性人都會期望的相同程度的謹慎來行事。這是一個非常廣泛的標準。審慎查核原則是盡職調查的一個更具體的組成部分，它規定被分配責任的個人應盡職盡責地準確、及時地完成它。

14. Roger 最近在一家公司接受了一個新的安全專業人員職位，該公司在 IaaS 環境中運行其整個 IT 基礎設施。以下哪項最可能是 Roger 公司的責任？

  1. 配置網路防火牆
  2. 應用 hypervisor 更新
  3. 修補操作系統
  4. 在處置前擦除硬碟

答案： C

說明： 在基礎設施即服務環境中，供應商負責硬體和網路相關的責任。這些包括配置網路防火牆、維護 hypervisor 和管理物理設備。客戶保留對其虛擬機實例上操作系統的修補責任。

15. Candace 正在為她組織的文件伺服器設計備份策略。她希望每個工作日執行一次備份，並且佔用的儲存空間盡可能小。她應該執行哪種類型的備份？

  1. 增量備份
  2. 完整備份
  3. 差異備份
  4. 事務日誌備份

答案： A

說明： 增量備份提供了包含最小數據量的選項。在這種情況下，那將是自上次增量備份以來修改的數據。差異備份將備份自上次完整備份以來修改的所有數據，這將是一個相當大的數量。完整備份將包括伺服器上的所有資訊。事務日誌備份專門設計用於支持資料庫伺服器，對文件伺服器無效。

16. Alyssa 的團隊最近實施了一個新系統，該系統從各種日誌來源收集資訊，分析該資訊，然後觸發自動化 playbook 以響應安全事件。哪個術語最能描述這項技術？

  1. SIEM
  2. 日誌存儲庫
  3. IPS
  4. SOAR

答案： D

說明： 安全資訊和事件管理 (SIEM) 系統從多個來源關聯資訊並執行分析，但它們止步於提供自動化 playbook 響應。這是安全編排、自動化和響應 (SOAR) 平台的領域。入侵防禦平台範圍更有限，允許根據 IPS 本身執行的分析來阻止流量。日誌存儲庫僅收集日誌資訊，不執行分析。

17. Nick 正在評估他組織未來數據中心的選項。以下哪個選項通常會產生最大的前期成本？

  1. 主機托管設施
  2. 雲數據中心
  3. 本地數據中心
  4. SaaS 產品

答案： C

說明： 這是“購買與構建”決策的一個典型例子。任何時候組織選擇構建資本資源，例如本地數據中心，都會涉及非常高的前期投資。這裡的其他產品涉及利用他人進行的資本投資，前期成本較低（或沒有），但經常性成本較高。

18. Ben 是一位 IT 審計員，希望確保組織有適當的機制來為系統和應用程式創建適當的審計跟踪。以下哪個技術聚合和關聯日誌條目？

  1. SIEM
  2. IPS
  3. EDR
  4. CASB

答案： A

說明： 安全資訊和事件管理 (SIEM) 解決方案聚合來自許多不同來源的日誌條目，並將它們關聯起來以創建可解釋的審計跟踪。入侵防禦系統 (IPS) 分析和阻止可疑的網路流量。端點檢測和響應 (EDR) 平台監控端點是否存在惡意軟體並響應惡意軟體感染。雲訪問安全代理 (CASB) 在雲提供商之間執行組織的安全策略。IPS、EDR 和 CASB 解決方案是組織安全計劃的重要組成部分，但它們不聚合和關聯日誌條目。

19. 除了雲提供商實施的安全控制之外，雲客戶還必須考慮由 ___________________ 實施的安全控制。

  1. 各自的監管機構
  2. 終端用戶
  3. 雲客戶先前在本地環境中使用的任何供應商
  4. 提供商依賴的任何第三方

答案： D

說明： 由於供應鏈依賴性會影響服務，雲客戶將需要確保任何第三方依賴性是安全的。監管機構和終端用戶不提供企業安全性，因此選項 A 和 B 不正確。用於本地安全的供應商將不再影響數據，因此選項 C 不正確。

20. Brittney 正在審查她組織的災難恢復流程數據，並注意到該企業資料庫伺服器的 MTD 為 30 分鐘。她對該伺服器的 RTO 有何了解？

  1. 它需要少於 30 分鐘。
  2. 它需要至少 30 分鐘。
  3. MTD 太短，需要更長。
  4. RTO 太短，需要更長。

答案： A

說明： 當 Brittney 審查恢復時間目標 (RTO) 數據時，她需要確保組織能夠在基於 30 分鐘最大容忍停機時間 (MTD) 的中斷後不到 30 分鐘內恢復。

21. Cameron 擔心針對其公司主要 Web 應用程式的分佈式拒絕服務 (DDoS) 攻擊。以下哪個選項將提供對大規模 DDoS 攻擊的最大彈性？

  1. 實施 CDN。
  2. 增加 Web 應用程式伺服器集群中的伺服器數量。
  3. 通過公司的 ISP 簽訂 DDoS 緩解服務合同。
  4. 增加可從一個或多個 ISP 獲得的帶寬量。

答案： A

說明： 由主要提供商運營的內容分發網路 (CDN) 可以比任何其他解決方案更容易地處理大規模 DDoS 攻擊。通過 ISP 使用 DDoS 緩解技術是次有用的能力，其次是增加帶寬和增加 Web 應用程式伺服器集群中的伺服器數量。

22. John 的網路開始出現緩慢的症狀。他啟動了一個數據包捕獲工具，意識到網路正受到 TCP SYN 數據包的轟炸，並認為他的組織是拒絕服務攻擊的受害者。正在違反哪個資訊安全原則？

  1. 可用性
  2. 完整性
  3. 機密性
  4. 拒絕

答案： A

說明： 拒絕服務 (DoS) 攻擊旨在壓垮系統，直到它無法處理合法請求。此類攻擊的目的是拒絕合法用戶訪問系統，這是對可用性原則的違反。

23. Mike 最近實施了一個入侵防禦系統，旨在阻止常見的網路攻擊影響他的組織。Mike 正在推行哪種類型的風險管理策略？

  1. 風險接受
  2. 風險規避
  3. 風險緩解
  4. 風險轉移

答案： C

說明： 風險緩解策略試圖降低風險發生的概率和/或影響。入侵防禦系統試圖降低成功攻擊的概率，因此是風險緩解的例子。

24. 您正在嘗試確定您的組織必須在您的 BC/DR 活動中保護的關鍵資產。以下哪個產物對您的工作最有用？

  1. 定量風險分析
  2. 定性風險分析
  3. 業務影響分析
  4. 風險偏好

答案： C

說明： 業務影響分析 (BIA) 正是為此目的而設計的：確定組織內資產/資源/數據的關鍵路徑。它是用於制定 BC/DR 計劃的完美工具。風險分析選項和風險偏好選項可能會為 BIA 提供輸入，但它們不是用於確定保護 BC/DR 活動所需關鍵資產的工具。因此，選項 A、B 和 D 都是不正確的。

25. 主 HVAC 系統中的組件故障導致 Kim 管理的數據中心出現高溫警報。解決問題後，Kim 應考慮做些什麼來防止將來出現類似問題？

  1. 閉環冷卻器
  2. 冗餘冷卻系統
  3. 沼澤冷卻器
  4. 將數據中心搬遷到更冷的氣候

答案： B

說明： 設計良好的數據中心應對其基礎設施的每個關鍵部分具有冗餘系統和能力。這意味著電力、冷卻和網路連接都應是冗餘的。Kim 應確定如何確保單一系統故障不會使她的數據中心離線。

26. Joe 是一家基於雲的 ERP 系統的安全管理員。他正準備為幾位新員工創建帳戶。當他創建帳戶時，他應該給所有新員工什麼默認訪問權限？

  1. 只讀
  2. 編輯者
  3. 管理員
  4. 無訪問權限

答案： D

說明： 在這種情況下，Joe 應遵循最小特權原則。他應默認不應用任何訪問權限，然後根據業務需要為每個用戶提供執行其工作職責所需的必要權限。只讀、編輯者和管理員權限可能對這些用戶中的一個或多個是必要的，但這些權限應基於業務需要分配，而不是默認分配。

27. Jason 運營一個雲數據中心，並希望提高管理員以編程方式與管理平面上的後端解決方案互動的能力。他可以使用哪種技術來最好地實現此類自動化？

  1. CASB
  2. API
  3. Hypervisor

81. 您正在與職能團隊的領導者一起審查新數據中心的要求。討論集中在如果發生停機可能丟失的數據量上。哪個指標與此討論最直接相關？

  2. 恢復點目標
  3. 最大允許停機時間
  4. 平均故障時間

答案： B

說明： 恢復點目標 (RPO) 是衡量在不對組織造成無法彌補的損害的情況下可能丟失的數據量。數據複製策略將對此指標影響最大，因為策略的選擇將決定有多少最近的數據可用於恢復目的。恢復時間目標 (RTO) 是衡量組織在不造成無法彌補的損害的情況下可以承受的中斷時間。這可能會受到複製策略的影響，但不如 RPO 受到的影響大。選項 A 不正確。最大允許停機時間 (MAD) 是組織在停止運營之前可以承受的中斷時間。這不依賴於 RPO，數據複製策略對其影響不大。選項 C 不正確。平均故障時間 (MTTF) 是衡量資產預期壽命（通常是硬體）的指標，由製造商/供應商確定。數據複製策略對此毫無影響。選項 D 不正確。

  2. 身份驗證
  3. 授權
  4. 聯合

答案： C

說明： 授權是向用戶和其他安全主體授予資源訪問權限的過程。識別和身份驗證是整體身份和訪問管理 (IAM) 過程的一部分，授權也是如此，但它們不具體描述授予資源訪問權限。聯合是在組織之間進行 IAM 的一種手段；授權是一個更具體的答案，因此聯合不正確。

說明： 將備份保存在同一環境中可以輕鬆回滾到最後一個已知良好狀態或在輕微事件（例如某些 VM 中的惡意軟體感染）後重新實例化乾淨的 VM 映像。合規性的易用性不會由備份的位置決定，因此選項 A 不正確。差旅不應成為雲使用的主要成本；選項 B 不正確。備份的位置對用戶培訓沒有影響；選項 C 不正確。

100. 虛擬機 (VM) 配置管理 (CM) 工具應要求託管系統執行 ___________________。

  1. 生物識別識別
  2. 反篡改機制
  3. 日誌文件生成
  4. 回溯能力

答案： C

說明： 事件日誌記錄對於事件管理和解決至關重要；這可以設置為 CM 工具的自動化功能。並非所有系統都需要或可以使用生物識別；選項 A 不正確。通常，篡改是指對設備的物理入侵；由於問題是關於 VM 的，因此可能不適用。選項 B 不正確。回溯在許多司法管轄區是非法的；選項 D 不正確。
