第七章 練習測試 1（含答案與說明）

1. 您正在選擇一個數據中心環境來託管由您的組織運行的雲應用程式。您的主要要求是數據中心必須不需要為設備維護而關閉。根據 Uptime Institute 的分級系統，哪個是可接受的最低數據中心級別？

  1. Tier 1
  2. Tier 2
  3. Tier 3
  4. Tier 4

答案： C

說明： Tier 1 和 2 級數據中心不需要有足夠的冗餘組件以在設備維護期間保持運行。這是 Tier 3 和 Tier 4 級數據中心的要求。因此，最低可接受級別將是 Tier 3。

2. 在基礎設施即服務 (IaaS) 安排中，誰負責保護基於雲的應用程式的安全？

  1. 雲提供商
  2. 雲客戶
  3. 監管機構
  4. 終端用戶/客戶端

答案： B

說明： 在基礎設施即服務 (IaaS) 模型中，提供商僅負責配置設備和計算/儲存容量；客戶負責其他所有事情，包括應用程式的安全性。所有其他答案都是不正確的，因為這些個人/組織不承擔保護基於雲的應用程式安全的責任。

3. Brenda 的公司僱用了許多應用程式開發人員，他們創建軟體以滿足許多不同的業務需求。她正在著手一個驗證使用已驗證開源軟體的項目，並擔心這些開發人員在不知情的情況下使用了軟體庫。以下哪種技術最能幫助識別這些用途？

  1. 動態應用程式安全測試 (DAST)
  2. 靜態應用程式安全測試 (SAST)
  3. 軟體組成分析 (SCA)
  4. 交互式應用程式安全測試 (IAST)

答案： C

說明： 軟體組成分析 (SCA) 旨在識別組織內部使用的軟體中的開源庫和其他依賴項。它通常會檢測開發人員在軟體開發過程中可能不知道的依賴項。靜態應用程式安全測試 (SAST)、動態應用程式安全測試 (DAST) 和交互式應用程式安全測試 (IAST) 都是用於識別開發人員創建的代碼中安全缺陷的技術，不一定能檢測依賴包的使用。

4. ________ 是監視網路中惡意活動或策略違規並向管理站生成電子警報和/或報告的軟體或設備。

  1. 主機入侵檢測系統 (HIDS)
  2. 硬體安全模塊 (HSM)
  3. 網路入侵檢測系統 (NIDS)
  4. 虛擬專用網路 (VPN)

答案： C

說明： 網路入侵檢測系統 (NIDS) 專門用於在網路級別監視異常或惡意系統活動，並提供有關此類活動的警報和/或報告。它們本身不提供任何類型的阻止活動或補救措施。

5. Carla 在一家基礎設施即服務 (IaaS) 提供商工作。她正在分析用於多租戶環境的 hypervisor 的安全設置。誰應該有權修改這些 hypervisor 上的設置？

  1. 只有具有適當安全培訓和訪問權限的 Carla 公司的員工。
  2. Carla 公司的員工以及在該特定 hypervisor 上運行虛擬機的客戶。
  3. Carla 公司的員工以及具有適當安全培訓的客戶。
  4. 這些群體都不應具有 hypervisor 訪問權限。

答案： A

說明： Hypervisor 訪問應嚴格限制，以保護多租戶環境中的租戶隔離。因此，絕不應授予客戶對 hypervisor 本身的直接訪問權限。hypervisor 上任何必要的配置更改都應由 IaaS 提供商的員工進行。

6. 在哪個雲計算模型中，客戶與雲供應商的其他客戶共享計算基礎設施，而一個客戶可能不知道其他客戶的身份？

  1. 公有雲
  2. 私有雲
  3. 社群雲
  4. 共享雲

答案： A

說明： 在公有雲計算模型中，供應商構建一個單一平台，供許多不同客戶共享。這也稱為共享租戶模型。

7. 您在一個政府研究機構工作。您的組織經常與其他政府研究組織共享數據。您希望在這些組織之間創建單一登錄體驗，以便每個組織的用戶可以使用該組織頒發的用戶 ID/身份驗證登錄，然後訪問所有其他組織的研究數據。與其在每個其他組織複製每個組織的數據儲存（這是實現此目標的一種方法），您希望每個用戶都能訪問每個組織的特定儲存資源。
這種安排的術語是什麼？

  1. 公鑰基礎設施 (PKI)
  2. 可移植性
  3. 聯合
  4. 否認

答案： C

說明： 這就是聯合的定義。PKI 用於在不受信任的介質上建立各方之間的信任，可移植性是描述能夠將數據從一個雲提供商移動到另一個雲提供商的可能性的特性，否認是指交易的一方否認參與了該交易。

8. Paul 的組織維護受 HIPAA 管轄的受保護健康資訊。他希望添加一個新的安全控制，能夠檢測員工何時試圖有意或無意地從其組織的系統和網路中移除此敏感資訊。哪個安全控制最能滿足他的需求？

  1. DLP
  2. CASB
  3. IPS
  4. NGFW

答案： A

說明： 所有這些技術都有一定的能力來協助完成此任務，但問題是關於最適合此目的的技術。數據丟失預防 (DLP) 系統專門設計用於檢測和防止敏感資訊的未經授權洩露。雲訪問安全代理 (CASB) 旨在跨雲提供商一致地執行安全策略。入侵防禦系統 (IPS) 旨在識別和阻止網路或系統上的惡意活動。下一代防火牆 (NGFW) 旨在阻止未經授權的網路連接。

9. 以下哪項 *不是* 定義雲計算的基本要素？

  1. 廣泛的網路訪問
  2. 計量服務
  3. 異地儲存
  4. 按需自助服務

答案： C

說明： 雖然異地儲存是一種雲服務產品，但它不是雲計算定義的內在要素。雲計算的核心特性確實包括廣泛的網路訪問、計量服務和按需自助服務。

10. Justin 最近參加了一次災難恢復計劃測試，團隊坐在一起討論了對一個場景的響應，但實際上沒有激活任何災難恢復控制措施。他參加了哪種類型的測試？

  1. 清單審查
  2. 完全中斷測試
  3. 並行測試
  4. 桌面演練

答案： D

說明： 在桌面演練期間，團隊成員聚集在一起演練一個場景，而不對信息系統進行任何更改。清單審查是干擾最小的災難恢復測試類型。在清單審查期間，團隊成員各自審查其災難恢復清單的內容，並提出任何必要的更改。在並行測試期間，團隊實際激活災難恢復站點進行測試，但主站點仍保持運行。在完全中斷測試期間，團隊關閉主站點並確認災難恢復站點能夠處理常規操作。完全中斷測試是最徹底的測試，但也是干擾最大的測試。

11. Tonya 受僱於一家雲服務提供商，負責評估該提供商的安全計劃。哪個 ISO 標準最能幫助她確保其組織擁有一套健全的安全控制措施？

  1. ISO 27001
  2. ISO 27002
  3. ISO 27017
  4. ISO 27701

答案： C

說明： 這是一個非常困難的問題，因為它引用了一些類似的 ISO 標準。我們可以立即排除 ISO 27701 作為選項，因為該標準涉及隱私控制，而不是 Tonya 正在評估的安全控制。其餘三個標準都涵蓋安全控制。然而，ISO 27017 專門針對雲提供商安全控制，將是最適合 Tonya 需求的標準。ISO 27001 和 27002 是更通用的標準，涵蓋任何類型組織的安全控制。

12. Katie 的組織最近遭受了一次數據洩露，暴露了一個包含學生記錄的資料庫。這些記錄除了標記化的學生 ID 號碼外，不包含任何身份識別信息。以下哪個陳述最準確地反映了 Katie 對這些記錄暴露的感受？

  1. 只有當標記化過程中的數據是安全的時，數據才是安全的。
  2. 只有當密碼金鑰未被暴露時，數據才是安全的。
  3. 只有當標記化過程中使用了強哈希函數時，數據才是安全的。
  4. 數據不安全，應被視為已洩露。

答案： A

說明： 標記化用唯一的標識符替換敏感數據元素。這些記錄可以使用查找表在資料庫中重新識別。如果此查找表被洩露，則數據不安全。如果查找表保持安全，則數據的隱私保持不變。標記化不使用密碼金鑰，密碼金鑰用於加密。標記化也不使用哈希函數，哈希函數用於哈希處理。

13. Bob 正在設計一個數據中心以支持其組織，一家金融服務公司。以下哪個行動最能增強 Bob 在數據中心創建彈性的努力？

  1. 確保所有入口都使用基於生物識別的鎖進行保護。
  2. 從不同供應商處購買不間斷電源 (UPS)。
  3. 在所有管理員的人事審查中包括財務背景調查。
  4. 確保所有活動地板至少有 24 英寸的淨空。

答案： B

說明： 使用來自不同供應商的相同類型的多個系統可以增加彈性；如果一個產品存在固有的製造缺陷，另一個則不應存在，如果它來自不同的生產商。其他建議都是構建數據中心的良好實踐，但它們不提供冗餘或彈性。

15. A(n) ______________ 由一個看起來是網路一部分但實際上被隔離和監控的計算機、數據或網路站點組成。它似乎還包含有價值的數據或資源，但實際上是假的。

16. Rusty 正在評估一個基於 Web 的 SaaS 應用程式的安全性，並希望驗證該站點在 Web 伺服器和客戶端之間提供強加密。實現此目標的最常用方法是什麼？

  1. 安全套接字層 (SSL)
  2. DNS 安全擴展 (DNSSEC)
  3. 網際網路協議安全 (IPsec)
  4. 傳輸層安全 (TLS)

答案： D

說明： 傳輸層安全 (TLS) 是實現超文本傳輸協議安全 (HTTPS) 的最常用方法，它在 Web 伺服器和客戶端之間提供加密。安全套接字層 (SSL) 以前用於此目的，但現已被認為過時且不安全。網際網路協議安全 (IPsec) 用於在遠程用戶與網路之間或兩個網路之間創建加密網路連接，但不常用於 Web 應用程式。域名系統安全擴展 (DNSSEC) 用於提供經過身份驗證的名稱解析。

17. 您運營一個雲服務，並希望獲得一份報告，該報告確認您的安全控制的有效性，並提供您可以使用的有關控制差距的詳細資訊以進行補救。您應進行哪種類型的審計？

  1. SOC 1
  2. SOC 2
  3. SOC 3
  4. SOC 4

答案： B

說明： SOC 2 審計涵蓋資訊的機密性、完整性和可用性，僅供內部受眾使用，因為它們包含敏感資訊。SOC 2 審計只能在保密協議下與客戶共享。SOC 3 審計涵蓋與 SOC 2 審計相同的控制措施，但旨在供一般受眾使用。SOC 1 審計僅涵蓋與財務報表和報告相關的內部控制。SOC 4 審計不存在。

20. 在記錄有關網際網路用戶位置的資訊時，哪個來源提供最準確的物理位置數據？

  3. 用戶證明
  4. MAC 地址關聯

答案： A

24. 根據此場景中的資訊，火災對 Roscommon Cloud Solutions 數據中心的影響的暴露因子是多少？
（場景：Darcy 是 Roscommon Cloud Solutions 的資訊安全風險分析師。她目前正在決定公司是否應為其主要數據中心購買升級的滅火系統。數據中心設施的重置成本為 200 萬美元。在諮詢了精算師、數據中心經理和消防主題專家後，Darcy 確定一次典型的火災可能需要更換建築物內的所有設備，但不會造成重大的結構損壞。他們共同估計，從火災中恢復的成本為 750,000 美元。他們還確定，公司可以預期每 50 年發生一次這種規模的火災。）

  1. 7.5%
  2. 15.0%
  3. 27.5%
  4. 37.5%

答案： D

說明： 暴露因子是風險管理人員預期如果風險發生時設施損壞的百分比。它的計算方法是用損壞金額除以資產價值。在這種情況下，即 750,000 美元的損壞除以 200 萬美元的設施價值，即 37.5%。

25. 根據此場景中的資訊，Roscommon Cloud Solutions 數據中心發生火災的年度發生率是多少？（場景：參考問題 24 的場景）

  1. 0.002
  2. 0.005
  3. 0.02
  4. 0.05

答案： C

說明： 年度發生率是風險分析師預期每年風險發生的次數。在這種情況下，分析師預計每 50 年發生一次火災，即每年 0.02 次。

26. 根據此場景中的資訊，Roscommon Cloud Solutions 數據中心發生火災的年度損失預期是多少？（場景：參考問題 24 的場景）

  1. $15,000
  2. $25,000
  3. $75,000
  4. $750,000

答案： A

說明： 年度損失預期是通過將單次損失預期 (SLE) 乘以年度發生率 (ARO) 來計算的。在這種情況下，SLE 為 750,000 美元，ARO 為 0.02。將這些數字相乘得到 ALE 為 15,000 美元。

27. 在大量使用許多不同雲服務的組織中，電子發現工作的最大障礙是什麼？

  1. 識別相關記錄
