第五章 問題：雲安全運營（含答案與說明）

1. 哪個術語用於描述 IT 服務提供商和客戶之間描述服務級別目標以及客戶和提供商責任的協議？

  1. OLA
  2. SAC
  3. SLA
  4. SLR

答案： C

說明： 服務級別協議 (SLA) 定義了服務級別目標以及 IT 服務提供商和客戶的責任。OLA（運營級別協議）是 IT 服務提供商與同一組織的另一部分之間的內部協議，支持服務提供商的服務交付。服務驗收標準 (SAC) 是用於確定服務是否滿足其質量和功能目標的標準。最後，服務級別要求 (SLR) 從客戶的角度定義了服務的要求。

2. Sally 正在制定她組織的溝通計劃，並且知道客戶是該計劃中需要包含的重要群體。積極的客戶溝通有助於哪個關鍵功能？

  1. 違規通知
  2. 監管合規性
  3. 管理期望
  4. 問題管理

答案： C

說明： 積極的客戶溝通是管理期望的關鍵。被動溝通通常用於數據洩露通知、監管合規性和問題管理。

3. Juanita 在她新安裝的 Linux 系統上發現了意外運行的程序，該系統是使用她的雲提供商的自訂 Linux 發行版構建的，但尚未允許來自網際網路的連接。最可能的原因是什麼？

  2. 安裝時，該版本的 Linux 會自動下載幫助代理。
  3. 雲供應商通常會在自己的發行版中安裝幫助程序。
  4. 攻擊者安裝了應用程式。

答案： C

說明： Juanita 知道主要的雲供應商會提供他們自己的 Linux 定制版本，其中通常包括額外的代理和工具，以幫助它們更好地與供應商的基礎設施協作。她應驗證情況是否如此，但這是對新構建系統描述的最可能情況。

答案： A

說明： 在雲中大規模管理系統時，Ben 知道最好的選擇通常是使用雲 IaaS 供應商的工具，特別是因為它們通常設計用於處理可能具有在供應商環境中工作的特殊功能的操作系統以及應用程式。

5. Jason 的組織被要求向其行業的監管機構提供有關其雲運營環境的資訊，包括年度審計資訊。當監管機構要求對其託管環境進行安全審計時，他最有可能向監管機構提供什麼？

9. Zoe 希望加快其傳統的發布管理流程。哪個現代方法最適合基於 ITIL v4 的快速發布導向組織？

12. Kathleen 希望集中化她的日誌捕獲和分析能力，並使用自動化工具來幫助她識別可能的安全問題。她應該尋找哪種類型的工具？

  1. SIEM
  2. IPS
  3. CASB
  4. MITRE

答案： A

說明： Kathleen 應尋找安全資訊和事件管理 (SIEM) 工具。它們用於集中式日誌收集、分析和檢測能力，並且通常具有查找問題並對其發出警報的自動化方法。IPS（入侵防禦系統）用於檢測和阻止攻擊，CASB（雲應用程式安全代理）用於控制和管理對雲服務的訪問，MITRE 是一個美國政府資助的研究組織，重點關注安全工作。

13. Elaine 希望確保傳輸中的流量已加密。通常使用哪種技術來保護傳輸中的數據？

  1. VLAN
  2. TLS
  3. DNSSEC
  4. DHCP

14. Ujama 希望保護其環境中的系統免受通過 SSH 訪問。如果他需要為本地連接保留該服務，他應該怎麼做？

  1. 在他的防火牆上阻止到 TCP 端口 3389 的入站連接。
  2. 在他的防火牆上阻止到 TCP 端口 3389 的出站連接。
  3. 在他的防火牆上阻止到 TCP 端口 22 的入站連接。
  4. 在他的防火牆上阻止到 TCP 端口 22 的出站連接。

答案： C

說明： 阻止到端口 22（默認 SSH 端口）的入站連接將阻止來自網路外部的攻擊者和第三方訪問 SSH，只要它未更改到另一個端口。TCP 3389 與 RDP 相關。

23. 雲安全聯盟的雲事件響應 (CIR) 框架記錄了事件響應中客戶與雲提供商責任的典型劃分，包括指出在 SaaS 環境中，雲提供商幾乎對所有風險負責。在 IaaS 環境中，誰負責網路風險？

  1. 客戶
  2. 客戶和服務提供商雙方
  3. 服務提供商
  4. 第三方事件響應者

26. Kolin 需要從 Azure 託管的 VM 收集鑑識數據。在捕獲 VM 的操作系統和數據磁碟的磁碟快照後，他應如何驗證其鑑識數據？

  1. 比較 VM 的操作系統和數據磁碟的哈希值以及每個快照的哈希值。
  2. 製作快照的兩個副本並比較快照哈希值。
  3. 將 VM 導出為哈希值，然後驗證該哈希值。
  4. 將 VM 導出為磁碟映像並將磁碟映像的數字簽名與原始映像進行比較。

答案： A

說明： Azure 的最佳實踐建議為 VM 的操作系統和數據磁碟創建磁碟快照，安全地儲存快照，然後比較映像和原始映像之間的哈希值。比較快照副本的哈希值無法根據原始映像對其進行驗證，VM 無法導出為哈希值，磁碟映像未以對此類鑑識用途有意義的方式進行簽名。如果您想閱讀完整的實踐文件以獲取更多詳細資訊，可以在 https://docs.microsoft.com/en-us/azure/architecture/example-scenario/forensics 找到它。

98. 保管鏈文件記錄和跟踪有助於什麼？

  1. 不可否認性
  2. 貌似可信的否認
  3. 調查人員篡改數據
  4. 與執法部門接觸

答案： A

說明： 創建和維護適當的保管鏈文件記錄為流程提供了不可否認性，確保處理過程能夠經受住以後的審查。調查人員的數據篡改仍可能發生，但會在正確記錄的鏈中被識別出來。保管鏈可能由執法部門要求，但它無法幫助公司與他們接觸，貌似可信的否認不是一個鑑識概念。

99. Ben 希望在兩個網路段之間提供邏輯隔離。通常使用哪種技術來定義此目的的網路？

  1. DHCP
  2. VLAN
  3. VPN
  4. STP

答案： B

說明： VLAN（虛擬局域網）是用於隔離網路設備的邏輯覆蓋。DHCP（動態主機控制協議）提供了一種在請求時自動向系統提供 IP 地址和其他重要網路資訊的方法。VPN（虛擬專用網路）用於在通常通過不受信任網路的網路之間創建安全通道，STP（生成樹協議）用於防止網路中的環路。
